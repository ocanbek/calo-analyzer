# Screenshots

This folder is a placeholder for app screenshots (e.g. the upload screen,
the two Plotly charts, the summary table).

To add one: run the app (`streamlit run app/streamlit_app.py`), take a
screenshot, save it here (e.g. `docs/images/heat-flow-plot.png`), and
reference it from the main [README.md](../../README.md) with:

```markdown
![Heat flow plot](docs/images/heat-flow-plot.png)
```
